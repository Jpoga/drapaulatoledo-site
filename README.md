# Site Dra. Paula Toledo

Site estático desenvolvido em HTML, CSS e JavaScript, com:

- SEO técnico no HTML: title, meta description, canonical, Open Graph, Twitter Card, Schema.org Physician e FAQPage.
- Layout responsivo com menu mobile.
- Animações por IntersectionObserver e barra rolante de condições atendidas.
- FontAwesome via CDN para ícones públicos.
- Imagens da identidade visual em PNG/WebP na pasta `assets/img`.
- WhatsApp flutuante disponível em todas as páginas.

## Como publicar

Envie todos os arquivos para a raiz da hospedagem. Substitua o domínio do `canonical`, Open Graph e Schema.org pelo domínio oficial antes de publicar.

## Estrutura

- `index.html`
- `assets/css/styles.css`
- `assets/js/main.js`
- `assets/img/`
